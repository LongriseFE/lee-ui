# lee-ui
lee-ui是一个vue的常用组件库，目前正值起步，项目有待完善！
## 安装
``` bash
yarn add lee-ui // npm install lee-ui --save
```
## 使用
``` javascript
import LeeUI from 'lee-ui'
import '../packages/theme-default/lib/index.css'
Vue.use(LeeUI)
```
